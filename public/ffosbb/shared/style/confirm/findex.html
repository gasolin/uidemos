Confirm

Used for

  • Prompts user to take action. eg

  • Confirm a deletion

  • Respond to a system event (eg: restart device after a SIM card swap)

  • Grant or deny a permission

Example

  • Default option

  • Confirm with content

Characteristics

  • Modal: occupies the screen and requires user input to clear.

  • Title

  • Body (optional)

  • Icon (optional)

  • Input: Confirmation. (Can customize label string)

  • Input: Cancel. (Can customize label string)
